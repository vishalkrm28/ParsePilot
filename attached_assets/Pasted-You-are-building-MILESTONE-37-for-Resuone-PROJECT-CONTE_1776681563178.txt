PRIMARY GOAL
Create a workflow where the user can:
1. connect calendar and email providers later-ready
2. manage reminders inside Resuone first
3. receive smart reminders for applications and interviews
4. sync interview events to Google Calendar
5. create Gmail-ready email drafts or save sync-ready outbound records
6. get AI-suggested reminder timing and follow-up timing
7. centralize all reminders, upcoming events, and action prompts

USER JOURNEY
1. User opens an application
2. User sees:
   - follow up due soon
   - interview coming up
   - no thank-you email drafted yet
   - reminder suggestions
3. User can:
   - create reminder
   - accept AI-suggested reminder
   - sync interview to calendar
   - create Gmail-ready draft metadata
4. User opens a notifications center
5. User sees:
   - upcoming reminders
   - overdue actions
   - interview events
   - follow-up recommendations
6. User marks items complete, snoozes them, or converts them into calendar/email actions

IMPORTANT RULES
- Build internal reminder system first
