Build the complete MVP of a production-ready mobile app called **Tinkle Map**.

## Product
Tinkle Map is a women-first AI wrapper mobile app that helps users find **safe, clean, trusted public toilets around the world**. It is designed especially for women and young girls in real-life situations such as:
- road trips
- nights out
- travel
- festivals
- emergencies
- unfamiliar cities

This is a **mobile app**, not a web app.

## Core promise
This app should not just show toilets on a map.
It should help users decide **which toilet feels safest to use** based on trust signals, community reports, recent reviews, and AI-assisted recommendations.

## Primary goal
Build a polished, mobile-first MVP that feels real, usable, and launchable.

Users should be able to:
1. View nearby toilets on a map
2. See trust and safety-related scores
3. Open a toilet detail page
4. Mark toilets as Safe / Use With Caution / Avoid
5. Submit reviews and quick reports
6. Search in natural language with AI
7. Use emergency mode to find the nearest trusted toilet quickly
8. Save places
9. Create account and track contributions

## Tech stack
Use:

## Trust score engine
Build a simple but meaningful trust score system.

### Trust score should consider
- safety ratings
- cleanliness ratings
- lighting ratings
- privacy ratings
- accessibility ratings
- Safe / Caution / Avoid votes
- number of reviews
- recency of reviews
- recent “closed/unavailable” reports

### Logic guidance
- newer reviews should have more weight
- stale data should decay over time
- multiple Avoid votes should lower score
- multiple Safe votes should improve confidence
- recent verified positive reports should improve confidence
- low data listings should show lower confidence

### Display
For each toilet show:
- overall trust score
- confidence level:
  - High confidence
  - Medium confidence
  - Low confidence
- recently verified badge
- last confirmed usable timestamp
